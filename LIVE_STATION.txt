MeshRadio 24/7 Live Test Station Now Available!

We're excited to announce a continuously running test station for the MeshRadio community! You can now experience MeshRadio's stable, high-quality audio streaming without setting up your own broadcaster first.

Station Details:
  IPv6: 206:6da3:9f2:60d2:9769:21c6:10fe:11d4
  Port: 8799
  Format: 48kHz stereo, Opus 128kbps
  Content: Curated music playlist (continuous loop)
  Uptime: 24/7

How to Connect:

Via Terminal:
  ./meshradio
  Select "3" for Listener
  Enter IPv6: 206:6da3:9f2:60d2:9769:21c6:10fe:11d4
  Enter port: 8799

Via Web GUI:
  ./meshradio --gui --callsign YOUR-CALLSIGN
  Open http://localhost:8899
  Click "Start Listening"
  Paste IPv6: 206:6da3:9f2:60d2:9769:21c6:10fe:11d4

You should hear music within 1-2 seconds! This station demonstrates MeshRadio's production-quality continuous broadcasting, seamless song transitions, and multi-hour stability achieved in v0.3-alpha.

Prerequisites: You need Yggdrasil installed and running to access this station over the mesh network. For local testing without Yggdrasil, you can still broadcast and listen on localhost (::1).

Download MeshRadio: https://github.com/immartian/meshradio
